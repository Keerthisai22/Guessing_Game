# Number Guessing Game (Python)

A simple Python game where the user guesses a randomly generated number between 1 and 10.

## Features
- Random number generation
- Unlimited guesses until correct
- Clear feedback for each guess

## Technologies Used
- Python 3
- random module

## How to Run
1. Download or clone the repository
2. Run the program:
   python number_guessing_game.py
